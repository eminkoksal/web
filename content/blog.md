## Content & Structure Notes

- **Post template** *(for the new site builder):* each post should have **title · date · topic chip · reading time · language tag (EN/TR)** in its header metadata, plus an excerpt (~25–60 words) used in feeds.
- **No fake post list.** Do not seed this page with placeholder titles. The new site should pull the real WordPress posts (or migrated equivalents) into the feed when it launches. The pinned "Featured Post" can stay empty if there is no clear featured piece yet.
- **Migration note:** the live WordPress site at eminkoksal.com has the existing post archive. When migrating, decide whether each post (a) stays in its original language, (b) gets a translation, or (c) is retired. A spreadsheet pass before migration is faster than per-post decisions during it.
- **Topic taxonomy:** keep it short (≤8 topics). If the list grows past ~12, prune — long taxonomies dilute the brand and confuse readers.
